# Datasheet for the Synthetic mPTP Reconstitution Dataset

## Motivation

This datasheet documents the synthetic dataset generated in the mPTP reconstitution proof-of-concept notebook. The dataset was created to support computational exploration of candidate mitochondrial permeability transition behavior in a reconstitution-inspired framework. It is intended to help prioritize future biochemical experiments rather than to substitute for experimental data.

The motivation for creating the dataset is the continuing uncertainty around the molecular identity of the mitochondrial permeability transition pore. Since multiple protein systems, especially ANT, F1FO-ATP synthase, and Cyclophilin D, have been implicated in pore formation or regulation, a synthetic dataset enables structured comparison of how these candidate components might interact across disease-relevant stress conditions.

## Composition

The dataset consists of simulated mitochondrial observations across four biological state classes:
- cardiac control
- cardiac ischemia-reperfusion
- cancer therapy-naive
- cancer stress-adapted

Each state is crossed with five perturbation conditions:
- wild type
- CypD-low
- ANT-low
- ATPsyn-low
- ANT_ATPsyn_low

The full dataset contains repeated simulated observations for each state-perturbation combination, resulting in a balanced synthetic design space.

## Data generation process

Each observation is generated by sampling mitochondrial variables from clipped probability distributions defined for each state. These distributions reflect biologically plausible ranges rather than direct empirical measurements. State definitions differ in baseline membrane potential, calcium burden, oxidative stress, pH, energetic imbalance, cardiolipin integrity, Cyclophilin D activity, ANT function, ATP synthase pore propensity, and phosphate carrier modulation.

Derived variables are then computed from weighted combinations of these features. These derived variables include transient pore behavior, sustained pathological opening probability, calcium retention capacity-like behavior, threshold calcium for catastrophic opening, leakage burden, conductance-like signal, ATP preservation, mtDNA release proxy, and viability proxy.

Perturbation conditions are modeled by reducing the activity or pore-related contribution of specific candidate components. The combined ANT plus ATP synthase perturbation is designed to represent a cooperative reduction in pore competence.

## Source data

There are no real source records in this dataset. No patient samples, animal measurements, mitochondrial isolates, omics profiles, or proprietary datasets were used to populate the table. The dataset is fully synthetic.

The biological rationale used to choose variable names, ranges, and directional effects is informed by the peer-reviewed mitochondrial permeability transition literature, especially studies discussing Cyclophilin D, ANT, ATP synthase, ischemia-reperfusion injury, and mitochondrial adaptation in cancer.

## Unit of observation

Each row represents one synthetic mitochondrial condition instance under a defined biological state and perturbation context.

## Variables

### Core state variables
- `state`: biological context label
- `perturbation`: perturbation regime label
- `delta_psi_mV`: membrane potential proxy in millivolts
- `matrix_ca_uM`: matrix calcium proxy in micromolar units
- `mito_ros_au`: mitochondrial reactive oxygen species proxy in arbitrary units
- `pH_matrix`: matrix pH proxy
- `adp_atp_ratio`: energetic stress proxy
- `cardiolipin_index`: membrane lipid integrity proxy
- `cypd_activity`: Cyclophilin D activity proxy
- `ant_function`: ANT function proxy
- `atpsyn_pore_propensity`: ATP synthase-linked pore propensity proxy
- `pic_modulation`: phosphate carrier modulation proxy

### Derived features and outcomes
- `transient_pore_score`: continuous score for transient pore-prone behavior
- `sustained_pore_prob`: probability-like score for sustained pathological opening
- `crc_au`: calcium retention capacity-like proxy
- `threshold_ca_uM`: threshold calcium proxy for sustained opening
- `leakage_auc`: permeability burden proxy
- `conductance_like`: channel-like activity proxy
- `time_to_sustained_s`: latency proxy to sustained opening
- `ATP_index`: bioenergetic preservation proxy
- `mtDNA_release_proxy`: damage-associated mitochondrial DNA release proxy
- `viability_proxy`: global functional preservation proxy
- binary classification target derived from sustained pathological opening threshold

## Recommended uses

The dataset is suitable for:
- hypothesis generation
- exploratory data analysis
- machine learning demonstrations on synthetic mitochondrial data
- stress-state comparison
- counterfactual perturbation analysis
- planning future in vitro reconstitution experiments

## Non-recommended uses

The dataset should not be used for:
- clinical inference
- translational claims without validation
- benchmarking real-world biological performance
- training models intended for direct deployment on experimental or patient samples without domain adaptation and validation
- making claims that a specific pore structure has been experimentally confirmed

## Distributional considerations

Because the dataset is simulated from designed state distributions, it is balanced by construction and may not reflect the prevalence of biological states in real experiments or disease populations. Correlations between features are partly induced by the generation equations and do not necessarily match the full covariance structure of real mitochondrial systems.

## Known limitations

Several limitations should be considered:
- variables are proxies, not direct experimental measurements
- parameter ranges are literature-informed but not derived from a single harmonized dataset
- the data-generating equations impose simplified relationships among calcium, ROS, pH, membrane potential, energetic stress, and candidate protein activities
- the dataset does not capture assay-specific noise, reagent variability, purification artifacts, or structural heterogeneity of membrane protein assemblies
- low-conductance and high-conductance pore behaviors are abstracted rather than modeled through mechanistic kinetic transitions

## Validation status

The dataset has internal consistency within the notebook and supports coherent machine learning and multivariate analyses. However, it has not been externally validated against experimental reconstitution data.

## Preprocessing

The notebook performs standard train-test splitting and machine learning preprocessing as required by the chosen models. Users who export the dataset for other purposes should document any additional normalization, transformation, thresholding, or feature engineering they perform.

## Licensing and reuse considerations

Because the dataset is synthetic and generated within the notebook, reuse is generally straightforward for academic, educational, and exploratory purposes. Users should preserve attribution and clearly state that the dataset is synthetic and not experimentally observed.

## Update policy

If future versions of the notebook change variable definitions, disease states, perturbation logic, or outcome equations, this datasheet should be revised to reflect the updated dataset design.
